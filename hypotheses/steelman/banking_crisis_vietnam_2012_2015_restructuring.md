# Steelman — Vietnam 2012-2015 banking restructuring

## The strongest opposing argument

Vietnamese reported NPL data is systematically understated relative to international standards (Cao-Pham-Nguyen 2017 estimates true NPL > 8% at peak), so the "4%" threshold is a soft test. The zero-VND takeovers and VAMC are effectively bank cleanup-by-administrative-fiat, which works in a state-controlled banking system in a way it would not in a market-priced one — generalising the case is hard.

## What the framework still captures

The hypothesis tests whether the canonical "managed restructuring without macro recession" pattern is observable using publicly available WDI data. If the threshold is too soft, that itself is a finding (suggesting Laeven-Valencia methodology should re-examine the case). The negative-control on real GDP (>= 5% growth maintained) is a real test: if the restructuring had failed, growth would have plausibly slipped.
