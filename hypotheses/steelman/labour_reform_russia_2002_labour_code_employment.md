# Steelman — Russia 2002 Labour Code

The strongest pro-reform reading: the 2002 Labour Code is widely credited within Russian-economist circles (Kapeliushnikov, Gimpelson) with smoothing the post-1998 labour-market recovery by codifying flexible practices that had emerged informally during the 1990s. The Code legitimised wage-arrears resolution, permitted broader use of fixed-term contracts, and rationalised firing procedures — reforms that arguably enabled the 2002-2007 formal-employment growth that the spec's null-leaning prior dismisses. Donor-pool comparison cannot recover the counterfactual because Ukraine and Belarus did not adopt comparable reforms and lagged on labour formalisation in ways that would inflate the apparent Russia gain. The oil-channel control test is the discipline: if the formal-employment gap is statistically distinguishable from zero only when the oil-price control is omitted, the commodity-channel interpretation wins. If the gap survives with the control, the Code-channel claim is supported. A v2 should use within-Russia regional variation in initial labour-market formality to identify Code-effect heterogeneity.
