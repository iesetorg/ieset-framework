# Steelman — Turkey 2001 banking crisis

## The strongest opposing argument

Turkey 2001 is sometimes argued to be primarily a fiscal-dominance / political-stalemate event (the November 2000 / February 2001 political crisis between president and prime minister triggered the run on the lira), with the banking-system fragility as a reinforcing rather than initiating channel. Coding this as a "banking crisis" risks downplaying that the BRSA had been created in 1999-2000 to clean up exactly the bank-balance-sheet rot that the 2001 episode then crystallised — the regulatory-reform sequencing matters.

## What the framework still captures

The 4-of-5 multi-metric threshold permits the joint test (currency, output, banking coding, programme, inflation) without requiring that any single mechanism dominate. The hypothesis tests whether the canonical multi-metric signature held, not whether banking dysfunction was the primary causal driver.
