# Steelman — Vietnam Labour Code 2019

The strongest counter-reading: Vietnam's 2021-2024 manufacturing-employment expansion is essentially the China+1 supply-chain diversification trade — Apple, Samsung, and other multinationals shifted production to Vietnam during the same period for trade-policy reasons unrelated to the 2019 Labour Code. The donor pool of ASEAN peers does not include any country that captured the same scale of supply-chain relocation, so the synthetic counterfactual cannot recover the trade-shock-free trajectory. The Code's most-touted ILO-conformity element (multiple-union recognition) has not yet been operationalised in practice — VGCL retains effective single-union status, undermining the institutional-quality channel. ILO Vietnam evaluations 2023 acknowledge that ratification of ILO C-87 (freedom of association) remains pending, weakening the structural-reform interpretation of the Code. A clean refutation pathway: a sector decomposition that shows the formal-employment gain is concentrated in Vietnamese-domestic-firm manufacturing (where the Code bites) rather than FDI manufacturing (where supply-chain effects dominate) would support the labour-reform interpretation; the opposite pattern would refute it.
