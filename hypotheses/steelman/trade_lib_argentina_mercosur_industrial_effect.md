# Steelman — Argentina Mercosur industrial effect

## The strongest opposing argument

1. **Macro instability dominates.** Argentina's 2001 collapse,
   2018-2024 inflation, and recurring currency crises swamp any
   small Mercosur effect. The descriptive null is essentially
   inevitable given the macro environment.

2. **Brazil-bilateral effect is positive.** Argentine bilateral
   manufactured-goods exports to Brazil rose substantially under
   Mercosur (autos especially); the aggregate manufacturing-share
   metric misses this bilateral composition shift.

3. **Comparator selection.** Chile (the canonical comparator) is
   a small open commodity-exporter; Mexico is dominated by
   NAFTA dynamics. The "non-Mercosur Latin American" donor pool
   is methodologically thin.

## What the framework still captures

The descriptive null is honest and well-calibrated. The PARTIAL
verdict path captures a positive Argentine performance if it
emerges. The bilateral-composition critique deserves a separate
spec on Argentine-Brazilian bilateral trade.
