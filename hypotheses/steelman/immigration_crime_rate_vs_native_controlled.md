# Steelman — Immigration crime rate vs native-born (controlled)

**The strongest arguments against the framework's "no elevated foreign-born crime after controls" framing:**

The framework's coding follows the mainstream criminology consensus (Light-Miller 2018; Bell-Fasani-Machin 2013). The opposing view has a real evidentiary base in specific contexts and deserves engagement.

1. **Specific-origin × specific-offence × specific-country combinations show elevations that survive standard controls.** German BKA data 2015-2022 shows MENA-origin young-male cohorts with elevated sex-offence arrest rates that persist after age-gender controls. Swedish BRÅ data shows similar patterns for some origin groups on violent offences. Dutch CBS Moroccan-origin second-generation data on property offences. These are narrow findings — not "immigrants commit more crime" — but they are not zero after controls. The framework should report them honestly rather than aggregate them away.

2. **Age-sex controls are partial.** Migration flows are not random samples of sending populations. Asylum cohorts specifically are young-male-heavy by selection, which inflates raw offending rates; standard age-sex controls partly but not fully address this because within-age-sex strata, selection effects on impulse control, trauma history, prospects in destination labour market, etc., are unmeasured. "Residual after age-sex-SES" is not necessarily zero; it's the unmeasured selection + destination-reception composite.

3. **Integration-policy-explains-it is a real finding but can be over-applied.** Nordic crime-rate-convergence data by cohort IS strong evidence that labour-market integration speed matters. But "integration policy would have fixed it" is an untestable counterfactual in destinations where integration has been attempted and failed (Sweden's järva suburbs, Netherlands' Vogelaarwijken). In those cases, the framework should admit that integration-as-moderator has empirical limits.

4. **Police-recorded rates vs self-report victimisation divergence.** UK Crime Survey shows foreign-born victims report somewhat higher victimisation, which could be consistent with either higher offending OR higher within-community victimisation concentration (which is a policy-actionable finding). The framework's primary outcome (police-recorded offending) is the most-reported metric but not the only one; the steelman asks for self-report triangulation.

5. **Second-generation convergence is NOT universal.** In Sweden and France some origin-group second-generation cohorts show LESS convergence than expected on labour-market + crime metrics. The "integration takes time" hypothesis is contested for specific groups where second/third-generation outcomes do not approach native baseline. The framework should not report this as "problem of immigration"; it should report it as "failure of destination-country integration institutions" — but it should report it.

6. **Aggregate vs tail risk.** Average statistics can be misleading for policy if specific low-frequency high-severity crime types (terrorism, mass violence) are concentrated in specific immigrant subgroups. European intelligence agencies' own data on Islamist-terror-suspect cohorts shows concentration among specific origin + radicalisation-pathway populations. Average crime rates are the correct primary statistic; tail-risk concentration is a secondary statistic the framework should report when relevant to policy questions (visa screening, surveillance priorities).

**How this should shape the v1 result card when run:**

- Report log-rate-ratio by offence type separately: violent, property, drug, public-order. Findings often go in different directions.
- Report first-generation vs second-generation separately, with convergence rate as a key statistic.
- Disaggregate by legal-status channel (work visa / family / asylum / undocumented) — conflating them produces misleading aggregates.
- When specific origin × destination cells show elevated rates surviving controls, REPORT IT and examine whether destination-country integration policy explains it (as the literature suggests) or whether the residual exceeds plausible integration-moderator range.
- Include victim-reported data (UK CSEW, Dutch VM) as triangulation where available.
- Tail-risk statistics (specific low-frequency high-severity offence types concentrated in specific cohorts) as secondary panel where policy-relevant.

The framework's value is NOT in giving pre-cooked answers. It's in applying discipline: what does the data, honestly controlled, actually show — including where it shows things that challenge any political prior.
